
# Simple Multi-Page Website Using HTML & CSS

This project focuses on building a **simple and responsive multi-page website**
using **HTML and CSS** as part of a **Web Development Internship – Task 4**.
The main objective of this task is to understand **page navigation, reusable CSS,
and proper folder structure** in a real-world website.

## Features
- Multiple HTML pages (Home, About, Courses, Contact)
- Common navigation bar across all pages
- Reusable CSS stylesheet
- Active navigation link highlighting
- Clean and professional layout
- Proper use of relative paths

## Technologies Used
- HTML5
- CSS3
- VS Code
- Browser Developer Tools

## Project Structure
```

project/
│
├── index.html
├── about.html
├── courses.html
├── contact.html
│
└── css/
└── style.css

```


## How to Run
1. Download or clone the repository  
2. Open the project folder  
3. Open the `index.html` file in any modern web browser  
4. Use the navigation menu to explore different pages  

## Learning Outcomes
- Understanding of **multi-page website structure**
- Proper use of **relative paths**
- Maintaining **design consistency across pages**
- Difference between **single-page and multi-page websites**
- Importance of **organized folder structure**
